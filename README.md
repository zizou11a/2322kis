# IMGVO
🔗 **Live site:** https://imgvo.com

Free image converter & compressor — 100% in your browser. No upload, no signup.

## Features
- Convert JPG, PNG, WebP, AVIF, HEIC and more
- Compress images up to 90%
- Crop, resize, rotate, watermark
- Works offline
- 100% private — no files uploaded to servers
